# Weather-Mobile-App

A simple, useful software weather app for Android. It gathers data from [OpenWeatherMap](https://openweathermap.org/), via their public API.

## Features
* Simple design
* Detailed 5 day forecast
* Works with any city in the world

## API
* Recycler Adapter
* Volley
* Piscasso
* ...
